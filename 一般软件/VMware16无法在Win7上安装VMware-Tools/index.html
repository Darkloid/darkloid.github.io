VMware16无法在Win7上安装VMware Tools

问题：
VMware 16 Player在Windows7虚拟机上安装VMware Tools，报错：

安装程序无法自动安装Virtual Machine Communication Interface(VSock)驱动程序。必须手动安装此驱动程序。

原因：
2019年12月微软更新了Windows驱动程序算法，2020年1月14日微软终止对Windows7系统的技术支持，VMware Workstation 16是2021年发布的，VM未支持旧版的Windows系统，另外Windows Vista系统也无法正常安装在VMware Workstation 15.5以上版本中。

解决方法：
在虚拟机上安装KB4474419补丁
下载地址：https://www.catalog.update.microsoft.com/Search.aspx?q=KB4474419
或直接下载64位iso格式：[windows6.1-kb4474419-v3-x64_b5614c6cea5cb4e198717789633dca16308ef79c.zip]

原文链接：https://baijiahao.baidu.com/s?id=1735850963168614527
